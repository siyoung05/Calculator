# 계산기

# 사용언어
## JavaFX + SceneBuilder

# 프로그램 설명
(1) Main파일을 실행시켜 첫 화면을 실행.

(2) 연산할 숫자 버튼을 눌러 추가 시킨 후 부호 버튼을 눌러 연산식을 만듬.

(3) 이퀄 버튼을 눌러 계산 결과를 출력시킨다.
